Создание приложения Windows Forms

Целью этой лабораторной является знакомство с элементами управления Windows Forms.

Основное назначение Windows Forms - создание графического пользовательского интерфейса (GUI).

Эта технология позволяет использовать следующие компоненты:

  • диалоговые окна
  • текстовые поля - TextBox
  • надписи - Label
  • списки - ListBox, ComboBox
  • кнопки - Button
  • многие другие элементы управления
По существу, эти элементы управления являются просто классами из библиотеки .NET Framework.

Для того, чтобы открыть панель Конструктор (Toolbox), нужно:

  • Выбрать в меню View пункт Toolbox. При этом слева появится панель Конструктор.
  • Открыть при помощи Обозревателя Решений (Solution Explorer) форму. Панель конструктор отобразит элементы, доступные для данной формы.
Панель Конструктор(Toolbox) в Visual Studio позволяет перетаскивать мышью элементы управления в основную форму приложения. После этого IDE автоматически добавит исходный код для создания и инициализации экземпляра соответствующего класса. Размеры и расположение добавленных элементов управления также можно менять мышью.

У каждого из элементов управления есть множество дополнительных свойств. Для того, чтобы работать с ними, нужно открыть панель свойств. Щелкните правой кнопкой по свободному месту на форме и выберите пункт Свойства (Properties).

Измените свойства формы:
Тип элемента Наименование свойства Значение Пояснение
WindowsForm (Name) ProductCatalog Используется для работы с формой из кода программы. Каждой форме соответствует класс. Поэтому ее имя состоит из двух частей:
  • Первая часть (Product) - Назначение формы
  • Вторая часть (Catalog) - Тип формы. В этом случае - "справочник".
StartPosition CenterScreen Расположение формы при запуске программы. Это значение - "в центре экрана".
Text Справочник товаров Текст, отображаемый в шапке формы.

Эта форма будет справочником товаров. Слева на ней будет находиться список наименований товаров. Справа - свойства того товара, который сейчас выбран в списке. Основные операции со справочником это изменение, добавление и удаление товаров.

Расположите на форме и настройте свойства остальных элементов управления.

Задание для самостоятельного выполнения

Создайте формы для справочников групп товаров, сотрудников и клиентов

О каждой группе товаров известно ее наименование.

О каждом клиенте известно:

  • Наименование (юридического лица) - Name
  • Расчетный счет - CheckingAccount
  • ИНН - INN

О каждом сотруднике известно:
  • Имя - FirstName
  • Фамилия - LastName
  • Отчество - FatherName
  • Пол - Gender
  • Дата рождения - BirthDate
  • Статус "в браке" - Marriage
  • Почтовый индекс - PostalIndex
  • Адрес - Address
  • Тип найма - HireType
  • Примечание - Notes
