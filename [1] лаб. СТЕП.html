Создание приложения Windows Forms

Целью этого раздела является знакомство с технологией Windows Forms для построения простой программы на языке C#. Основное назначение Windows Forms - создание графического пользовательского интерфейса (GUI).

Эта технология позволяет использовать следующие компоненты:

  • диалоговые окна
  • текстовые поля - TextBox
  • надписи - Label
  • списки - ListBox, ComboBox
  • кнопки - Button
  • многие другие элементы управления
По существу, эти элементы управления являются просто классами из библиотеки .NET Framework.

Для того, чтобы открыть панель Конструктор (Toolbox), нужно выбрать в меню View пункт Toolbox. images/toolbox.png Панель Конструктор(Toolbox) в Visual Studio позволяет перетаскивать мышью элементы управления в основную форму приложения. После этого IDE автоматически добавит исходный код для создания и инициализации экземпляра соответствующего класса. Размеры и расположение добавленных элементов управления также можно менять мышью.

У каждого из элементов управления есть множество дополнительных свойств. Для того, чтобы работать с ними, нужно открыть панель свойств. Щелкните правой кнопкой по свободному месту на форме и выберите пункт Свойства (Properties).

Измените свойства формы:

Эта форма будет справочником товаров. Слева на ней будет находиться список наименований товаров. Справа - свойства того товара, который сейчас выбран в списке. Основные операции со справочником это изменение, добавление и удаление товаров.

Значения свойства (Name) будет использоваться для работы с элементом из кода программы. Поэтому оно должно быть максимально информативным. В рамках дисциплины рассмотрен один из подходов к именованию: Имя состоит из двух частей. Первая часть - сокращение от типа элемента управления. Вторая часть - его назначение. У кнопок это связанное действие. У текстовых полей и надписей это отображаемое свойство предмета.

Задание для самостоятельного выполнения

Создайте формы для справочников клиентов и поставщиков.

О клиентах известно:

  • Наименование - Name
  • Адрес - Address
  • ФИО начальника - Chief
  • Расчетный счет - CheckingAccount
  • Город - City
  • ИНН - INN
  • ОКОНХ - OKONH
  • ОКПО - OKPO
О поставщиках известно:
  • Наименование - Name
  • Адрес - Address
  • Расчетный счет - CheckingAccount
  • Город - City
  • ИНН - INN
  • ОКОНХ - OKONH
  • ОКПО - OKPO
